# Siamese Cat Creative Club SEO and Performance Audit

Audit date: 17 July 2026

Site: https://creative.siamesecat.cafe

Languages: Thai default, English under `/EN/`

## Executive result

The site is technically crawlable and now has a strong application-level performance baseline. The main remaining SEO dependency is operational: verify the domain in Google Search Console, submit the sitemap, and request indexing for the principal Thai and English pages.

## Crawl and indexing

- Live homepage, `robots.txt`, and `sitemap.xml` return HTTP 200 to Googlebot.
- `robots.txt` allows public content and excludes `/admin` and `/api`.
- The sitemap contains Thai and English canonical URLs with reciprocal language alternatives.
- Canonical, `hreflang="th"`, `hreflang="en"`, and `hreflang="x-default"` are present on localized marketing and legal pages.
- Sitemap entries now include an accurate `lastmod` date.
- Confirmation and generated error pages use `noindex,nofollow`.
- Admin and API responses include `X-Robots-Tag: noindex, nofollow`.
- A public `site:` search returned no results during this audit. Google Search Console's Indexing report and URL Inspection tool are the authoritative next checks.

## On-page and structured data

- Public pages have page-specific titles, descriptions, one primary heading, and crawlable internal links.
- Parent signup now has a bilingual `<h1>`.
- Open Graph and Twitter image metadata use the 1200x630 social image.
- Marketing pages include JSON-LD for `LocalBusiness`, `WebSite`, and `WebPage`.
- Structured data identifies Siamese Cat Creative Club as operated by Siamese Cat Cafe Co., Ltd. (Thailand), with address, telephone, email, image, and price range.
- Build checks parse the JSON-LD and fail if required business fields, canonical links, language links, image dimensions, or robots policies regress.

## Performance and assets

- Replaced the multi-megabyte navigation logos with a 2.7 KiB 96px WebP.
- Added 480px and 720px responsive WebP variants for customer-facing photographs.
- Added `srcset`, `sizes`, intrinsic dimensions, asynchronous decoding, and appropriate eager/lazy loading.
- Added short immutable-style caching for site assets and safe short caching for legal documents.
- Corrected an insufficient color-contrast token and language-toggle accessible names.
- Preserved zero layout shift and zero blocking time in the measured pages.
- The full browser audit passed 52 Thai/English mobile and desktop checks with no broken images, horizontal overflow, or language-misaligned internal links.

## Required operational actions

1. Verify `creative.siamesecat.cafe` or the parent domain in Google Search Console.
2. Submit `https://creative.siamesecat.cafe/sitemap.xml` once.
3. Use URL Inspection to request indexing for `/`, `/creative`, `/playgroup`, `/little-explorer-program`, `/membership`, `/contact`, and their `/EN/` equivalents.
4. Test the deployed homepage with Google's Rich Results Test and confirm the LocalBusiness item has no critical errors.
5. Link this site from the Siamese Cat Cafe Google Business Profile and other accurate business profiles where appropriate.
6. Review Search Console indexing, Core Web Vitals, and search queries after Google has had time to recrawl the site.

## Residual technical items

- Real-user Core Web Vitals require field traffic over time; lab scores cannot replace Search Console field data.
- The build reports that Next.js 14.2.18 has a published security update available. Upgrade and regression-test Next.js as a separate maintenance change.
- Hostinger's live HTML currently uses a revalidation-oriented cache policy. Upstream response time was approximately 17-20 ms during the audit, so this was not a bottleneck.
